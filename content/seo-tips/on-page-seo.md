# On-Page SEO Optimization Guide

*Master the art of optimizing individual pages for maximum search visibility*

## üéØ Content Optimization Fundamentals

### Title Tag Optimization
- **Length**: Keep between 50-60 characters (512px width)
- **Keywords**: Include primary keyword near the beginning
- **Uniqueness**: Every page should have a unique title
- **Branding**: Include brand name when appropriate
- **Compelling**: Write for users, not just search engines

### Meta Descriptions
- **Length**: 150-160 characters for optimal display
- **Call-to-Action**: Include compelling reason to click
- **Keywords**: Naturally incorporate target keywords
- **Unique**: Every page needs a unique description
- **Preview**: Think of it as your search result advertisement

### Header Structure (H1-H6)
- **H1**: One per page, includes primary keyword
- **H2-H6**: Create logical content hierarchy
- **Keywords**: Sprinkle secondary keywords naturally
- **Readability**: Break up content for better user experience

## üìù Content Strategy

### Keyword Research & Implementation
```
Primary Keyword: Focus keyword for the page
Secondary Keywords: Related terms and variations
Long-tail Keywords: Specific, lower-competition phrases
LSI Keywords: Semantically related terms
```

### Content Quality Factors
- **Word Count**: 1,500+ words for competitive keywords
- **Depth**: Comprehensive coverage of the topic
- **Originality**: Unique insights and perspectives
- **Freshness**: Regular updates and improvements
- **E-A-T**: Expertise, Authoritativeness, Trustworthiness

### Content Structure
1. **Introduction**: Hook readers and set expectations
2. **Table of Contents**: For longer articles
3. **Subheadings**: Break content into digestible sections
4. **Bullet Points**: Highlight key information
5. **Conclusion**: Summarize and include call-to-action

### Image SEO
- **Alt Text**: Descriptive alternative text for all images
- **File Names**: Use descriptive, keyword-rich filenames
- **File Size**: Optimize for fast loading (<100KB when possible)
- **Format**: Use WebP or AVIF for better compression
- **Responsive**: Ensure images work on all devices

### Video Content
- **Transcripts**: Provide text versions for accessibility
- **Thumbnails**: Create compelling preview images
- **Schema**: Implement VideoObject structured data
- **Hosting**: Consider YouTube for broader reach
- **Loading**: Use lazy loading for performance

## üîó Internal Linking Strategy

### Link Architecture
- **Hub Pages**: Create pillar content linking to related topics
- **Contextual Links**: Link naturally within content
- **Anchor Text**: Use descriptive, keyword-relevant text
- **Link Depth**: Ensure important pages are easily accessible
- **Link Equity**: Distribute authority throughout site

### Navigation Optimization
- **Breadcrumbs**: Help users understand site structure
- **Related Posts**: Encourage deeper engagement
- **Site Search**: Make content easily findable
- **Categories**: Organize content logically

## üì± User Experience Factors

### Page Speed Optimization
- **Core Web Vitals**: Optimize LCP, FID, and CLS
- **Image Compression**: Reduce file sizes without quality loss
- **Code Minification**: Compress CSS, JavaScript, and HTML
- **Caching**: Implement browser and server caching
- **CDN**: Use content delivery network for global speed

### Mobile Optimization
- **Responsive Design**: Ensure perfect mobile experience
- **Touch Targets**: 44px minimum size for clickable elements
- **Font Size**: 16px minimum for body text
- **Viewport**: Proper meta viewport configuration
- **Speed**: Prioritize mobile page load times

### Schema Markup
```json
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Your Article Title",
  "author": {
    "@type": "Person",
    "name": "Author Name"
  },
  "datePublished": "2024-01-01",
  "dateModified": "2024-06-01"
}
```

### Open Graph & Social Media
```html
<meta property="og:title" content="Your Page Title">
<meta property="og:description" content="Your page description">
<meta property="og:image" content="https://example.com/image.jpg">
<meta property="og:url" content="https://example.com/page">
<meta property="og:type" content="article">
```

### Twitter Cards

## üéØ Page-Specific Optimization

- Strong headline with primary keyword

- Compelling headlines and copy
- Clear call-to-action
- Social proof elements
